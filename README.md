## Import Project with git and maven command line

* git clone https://github.com/mmsdivino/chavesufba.git
* cd chavesufba
* git branch -a
* git checkout dev
* git branch team_xxx_yourname

### Run Project 
```
mvn spring-boot:run
```

### Clean Project 
```
mvn clean package
```
